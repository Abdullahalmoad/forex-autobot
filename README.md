# منصة التداول الآلي - Forex AutoBot SaaS

نظام تداول آلي يربط أي حساب MT5 من أي وسيط (عبر MetaApi.cloud)، يحلل السوق، ويفتح صفقات
تلقائياً ضمن حدود مخاطرة صارمة يضبطها المستخدم، مع إشعارات تليجرام فورية.

## المعمارية

المستخدم (جوال/متصفح) → Frontend (PWA) → Backend API (Node.js/Express)
الباك اند يتصل بثلاث خدمات: MetaApi.cloud (لتنفيذ الصفقات بأي وسيط)، Risk Engine (لحماية الرصيد)، و Supabase (قاعدة البيانات + تسجيل الدخول). عند فتح صفقة يرسل إشعار عبر Telegram Bot.

**ليش هذا التصميم يناسبك (جوال بدون حاسبة):**
كل شي يشتغل على سيرفر سحابي (Railway أو Render مثلاً) - جهازك بس يفتح تطبيق الويب للمراقبة والتحكم. لا حاجة لـ VPS ويندوز ولا MT5 مثبت عندك.

## خطوات التشغيل

### 1. MetaApi.cloud
- سجل حساب على https://app.metaapi.cloud
- أنشئ API Token واحطه بـ METAAPI_TOKEN
- التسعير: حساب واحد مجاني، بعدها حسب عدد الحسابات المربوطة - راجع أسعارهم الحالية بموقعهم

### 2. Supabase
- أنشئ مشروع جديد
- شغّل ملف backend/src/db/schema.sql بمحرر SQL بلوحة Supabase
- فعّل Supabase Auth

### 3. تليجرام
- كلم BotFather@ وأنشئ بوت جديد، خذ التوكن

### 4. تشغيل الباك اند محلياً للتجربة
داخل مجلد backend: انسخ .env.example إلى .env واعبي القيم، ثم شغّل npm install ثم npm run dev

### 5. النشر (Deployment)
- الباك اند: Railway أو Render
- الفرونت اند: Vercel أو Netlify

## الوحدات المتبقية

1. محرك الاستراتيجيات (Strategy Engine)
2. Scheduler/Worker
3. صفحة تسجيل الدخول + ربط حساب وسيط
4. لوحة تحكم (Dashboard)
5. نظام اشتراكات

⚠️ مهم جداً قبل أي تشغيل بحساب حقيقي:
- اختبر أسبوعين على الأقل بحسابات Demo متعددة
- فعّل auto_disable_on_breach دايماً
- ابدأ بمخاطرة منخفضة جداً لأول مستخدمين حقيقيين
- راجع المتطلبات القانونية لتشغيل خدمة تداول آلي بالعراق/السعودية قبل الإطلاق التجاري
